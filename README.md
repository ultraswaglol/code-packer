# code-packer
Online Client-Side tool to safely pack code projects into one file for LLMs (ChatGPT, Claude, Gemini). Token counter and file tree included. (Онлайн Client-Side инструмент для безопасной упаковки кода проектов в один файл для LLM (ChatGPT, Claude, Gemini). Включает подсчет токенов и дерево файлов.)
